### Features
- User registration & JWT authentication
- Role-based access control (Admin, Seller, Buyer)
- Create & manage auctions (Admin/Seller)
- Real-time bidding using WebSockets
- Live auction countdown timer
- Real-time highest bid updates
- Simple bid history per auction
- Persistent storage (PostgreSQL or MongoDB)
- Basic input validation & security
- REST API endpoints
- Basic Swagger/OpenAPI documentation
-  Auction lifecycle management (pending, active, completed)
- Real-time notifications (outbid, winning bid)
- Seller dashboard (live auction stats)
- Buyer dashboard (active bids & history)
- Input validation & API rate limiting
- Unit & integration testing
- Dockerized services (API + DB)
- Predict likely winning bids
- Personalized auction recommendations

### Tech Stack
- **Backend:** Node.js (Express) and  FastAPI
- **Database:**  MongoDB
- **Real-time:** Socket.io or FastAPI WebSockets
- **Auth:** JWT + bcrypt
- **Cloud** Docker and CI/CD
